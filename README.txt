Description:
Extension pour WooCommerce qui vous permet d'activer des boutons de contact sur vos pages produits. Les clients peuvent utiliser le bouton Appeler pour vous appeler directement, le bouton WhatsApp pour vous envoyer des messages sur WhatsApp avec les détails du produit, et le bouton SMS pour vous envoyer un message SMS avec les détails du produit. Ces fonctionnalités peuvent vous permettre de réaliser plus de ventes.

== Installation ==

1. Téléchargez le plugin depuis l'interface WordPress ou depuis le dépôt WordPress.org.
2. Décompressez le fichier zip et téléversez le contenu dans le répertoire `/wp-content/plugins/` de votre installation WordPress.
3. Activez le plugin via le menu "Extensions" dans WordPress.

Fonctionnalités
---------------
- Activez des boutons de contact sur les pages produits.
- Personnalisez les numéros de contact WhatsApp, Appel et SMS.
- Sélectionnez les pays cibles pour l'affichage des boutons.
- Activez ou désactivez les boutons de contact selon vos besoins.

Questions fréquentes
--------------------
1. Comment configurer les boutons de contact ?
   - Après avoir activé l'extension, accédez à la page de configuration 'WooCommerce Contact' dans le menu d'administration. À partir de là, vous pouvez configurer vos numéros de contact et sélectionner les pays cibles.

2. Puis-je désactiver des boutons de contact spécifiques ?
   - Oui, vous pouvez désactiver des boutons de contact individuels tels que WhatsApp, Appel ou SMS à partir de la page de configuration.
